# Small Business Sales Analysis

## Project Overview
This project analyzes sales data for a small business to identify revenue drivers, customer patterns, product performance, and potential reporting issues.

The goal is to simulate a practical business analytics project: starting from raw sales data, cleaning it, analyzing it, and turning the results into a dashboard and business recommendations.

## Business Questions
- Which products generate the most revenue?
- Which products or categories are growing or declining?
- Which customers are most valuable?
- Which customers appear to be inactive?
- Are there missing, duplicate, or inconsistent records?
- Can the monthly reporting process be made more repeatable?

## Tools Used
- Excel / Power Query
- Python
- SQL
- Power BI

## Planned Workflow
1. Import and inspect raw sales data
2. Clean missing, duplicate, and inconsistent records
3. Create summary tables using Python and SQL
4. Build a Power BI dashboard
5. Translate findings into business insights
